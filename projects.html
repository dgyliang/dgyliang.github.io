Bossu bot is a bot made for a popular messaging app, Discord. The bot is built in discord.py, and is hosted on Heroku. It is based off of a simple discord bot that I built for my COGS 18 final project. This is mainly a fun little personal bot that I use for my personal use, as well as a fun little side hobby of mine as a way to learn how to code. I have learned many new concepts while working on Bossu bot, such as web-scraping and very specialized python libraries. Thus, this bot will constantly receive new and experimental features. Some features include a built in chat bot that uses AIML based off of A.L.I.C.E Bot, and uses web-scraping to dynamically search for answers it does not know how to answer. It also has other web-scraping commands to look up player stats for certain games... and much more!
